# Base

mvvm模式下的base

Add it in your root build.gradle at the end of repositories:

```groovy
    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
```

Add the dependency

```groovy
    dependencies {
        implementation 'com.github.ydslib:Base:1.0.0'
    }
```
